## 🚀 **Como Executar os Exemplos**

### 1. **Clone o Repositório**
```bash
git clone https://github.com/cortezvini97/react-appointment-calendar
cd react-appointment-calendar
```

### 2. **Instale Dependências**
```bash
cd example
npm install
```

### 3. **Execute o Servidor**
```bash
npm start
```

### 4. **Acesse no Browser**
```
http://localhost:3000
```
